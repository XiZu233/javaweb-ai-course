# 课程介绍与环境准备

## 课程目标

本课程是一套**可自学、可部署**的 JavaWeb + AI 双版本实训方案。通过两个递进版本的学习，帮助你从零掌握企业级 Web 开发技能，并在进阶版本中接触当下热门的 AI 应用开发。

- **版本 A（传统实训版）**：面向零基础，建立 Web 开发全局认知
- **版本 B（AI 增强实战版）**：面向进阶，产出简历级 AI 全栈项目

## 学习路线

```
阶段一：前端基础（2天）
    HTML/CSS/JS → Vue 快速入门 → Axios 前后端交互

阶段二：后端基础（4天）
    Maven → SpringBoot 分层架构 → HTTP 协议 → MySQL → MyBatis → 工程规范

阶段三：后端实战（6天）
    部门/员工 CRUD → 分页查询 → 动态 SQL → 文件上传 → 多表关联 → 事务/AOP → 全局异常

阶段四：后端进阶（2天）
    JWT 认证 → Filter/Interceptor → Sa-Token → AOP 原理 → 自动配置原理

阶段五：前端实战（4天）
    Vue 工程化 → ElementUI/Plus → 部门/员工页面 → 登录与路由守卫

阶段六：部署运维（2天）
    Linux 基础 → Docker → Docker Compose → CI/CD

阶段七：AI 功能专题（3天，版本 B 专属）
    NL2SQL → RAG 知识库 → AI 简历解析 → Prompt 工程
```

## 环境准备

### 必装软件

| 软件 | 版本 | 用途 |
|------|------|------|
| JDK | 8（版本A）/ 17（版本B） | Java 运行环境 |
| Maven | 3.8+ | 项目构建 |
| Node.js | 18+ | 前端运行环境 |
| MySQL | 8.0 | 数据库 |
| IDEA | 最新版 | Java 开发 IDE |
| VS Code | 最新版 | 前端开发 IDE |
| Git | 最新版 | 版本控制 |
| Docker Desktop | 最新版 | 容器化部署 |

### 版本 A 环境检查

```bash
java -version        # 需显示 1.8.xxx
mvn -v               # 需显示 3.8.xxx
node -v              # 需显示 v18.xxx 或更高
mysql --version      # 需显示 8.0.xxx
docker --version     # 确认已安装
git --version        # 确认已安装
```

### 版本 B 额外要求

```bash
java -version        # 需显示 17.xxx
redis-server --version   # Redis 服务端（或用 Docker 启动）
```

## 项目导入

1. 克隆仓库：
```bash
git clone https://github.com/XiZu233/javaweb-ai-course.git
cd javaweb-ai-course
```

2. 导入后端项目（IDEA）：
   - File → Open → 选择 `version-a/tlias-backend` 或 `version-b/tlias-pro-backend`
   - 等待 Maven 自动下载依赖

3. 导入前端项目（VS Code）：
   - 打开 `version-a/tlias-frontend` 或 `version-b/tlias-pro-frontend`
   - 执行 `npm install`

## 快速启动

### Docker Compose 一键启动（推荐）

```bash
# 版本 A
cd version-a
docker-compose up -d
# 访问 http://localhost

# 版本 B
cd version-b
docker-compose up -d
# 访问 http://localhost
```

### 本地开发启动

**版本 A**：
```bash
# 1. 启动 MySQL（确保已创建 tlias_db 并执行 SQL 脚本）

# 2. 启动后端
cd version-a/tlias-backend
mvn spring-boot:run

# 3. 启动前端
cd version-a/tlias-frontend
npm install
npm run serve
```

**版本 B**：
```bash
# 1. 启动 MySQL 和 Redis（可用 Docker）
docker run -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql:8.0
docker run -d -p 6379:6379 redis:7-alpine

# 2. 启动后端（参考 version-b 的模块结构）
cd version-b/tlias-pro-backend
cd tlias-pro-server
mvn spring-boot:run

# 3. 启动前端
cd version-b/tlias-pro-frontend
npm install
npm run dev
```
