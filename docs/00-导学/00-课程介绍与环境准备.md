# 课程介绍与环境准备

## 一、课程目标

本课程旨在帮助学习者从零开始掌握 JavaWeb 全栈开发，并最终能够独立开发一个完整的前后端分离项目。

课程提供 **双版本学习路径**：
- **版本A（传统实训版）**：适合零基础学员，使用经典技术栈理解 Web 开发核心原理
- **版本B（AI增强实战版）**：适合有编程基础的学员，使用企业主流技术栈并融入 AI 功能

## 二、你将学到什么

### 版本A技能树
- 前端：HTML/CSS/JS、Vue2、ElementUI、Ajax
- 后端：SpringBoot2、MyBatis、MySQL、JWT
- 工程化：Maven、Git、三层架构

### 版本B技能树
- 前端：Vue3、TypeScript、ElementPlus、Pinia、Vite
- 后端：SpringBoot3、MyBatis-Plus、Redis、Sa-Token
- AI：NL2SQL、RAG知识库、AI简历解析
- 运维：Docker、Docker Compose、CI/CD

## 三、环境准备

### 必需软件

| 软件 | 版本 | 用途 |
|------|------|------|
| JDK | 8（版本A）/ 17（版本B） | Java运行环境 |
| Maven | 3.8+ | 项目构建 |
| Node.js | 16+ | 前端开发 |
| MySQL | 8.0 | 数据库 |
| Docker | 最新版 | 容器化部署 |
| IDEA | 2022+ | 开发IDE |

### 快速验证

```bash
# Java
java -version

# Maven
mvn -v

# Node
node -v
npm -v

# MySQL
mysql --version

# Docker
docker --version
docker-compose --version
```

## 五、学习建议

1. **先原理后工具**：HTTP协议、三层架构、数据库设计等原理是根基
2. **边学边练**：每看完一章，立即动手实现对应功能
3. **善用AI工具**：遇到报错先尝试自己解决，再借助AI辅助
4. **做好笔记**：记录踩坑记录和解决方案，这是你的宝贵财富
